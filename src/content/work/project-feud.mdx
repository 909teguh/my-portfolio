## Latar Belakang

Pada tahun 2020 saya mendapat tawaran untuk mengerjakan project sebuah board game 
b bernama Feud: the sheers prophecy. Project ini adalah project pertama saya dalam 
mengerjakan sebuah board game dan saya sangat senang bisa mendapatkan kesempatan ini.

## Proses

Dalam project ini saya membuat artwork utama yang digunakan sebagai cover boxnya, kemudian
saya juga merancang play mat nya. Selain itu saya juga membuat beberapa artwork untuk 
kartu-kartu yang ada di dalam board game tersebut.

![gambar feud](/images/work/feud.jpg)

## Section Ketiga
Project ini selesai pada tahun 2021 dan project ini mulai dilaunching di sebuah platform crowdfunding bernama Kickstarter. 
dalam waktu yang singkat project ini berhasil mendapatkan pendanaan yang cukup besar dan project ini berhasil diproduksi.
